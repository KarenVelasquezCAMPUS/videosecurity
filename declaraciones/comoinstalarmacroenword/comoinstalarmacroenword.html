Video Security

Esta es una aplicación de video seguridad que utiliza HTML5, CSS3 y JavaScript. Demuestra cómo reproducir un video utilizando HLS (HTTP Live Streaming) e implementa medidas de seguridad básicas como CORS, Content Security Policy y HTTP Basic Authentication.
